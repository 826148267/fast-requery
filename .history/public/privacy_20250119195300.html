隐私政策

数据收集

URL Parameters Manager 不会收集任何个人信息。所有数据都存储在您的本地浏览器中。

数据存储

扩展程序仅使用 Chrome 存储 API 来保存您的参数预设。这些数据完全存储在您的浏览器中，不会上传到任何服务器。

权限使用

  • activeTab：用于读取和修改当前标签页的 URL
  • storage：用于保存参数预设

第三方服务

本扩展程序不使用任何第三方服务或分析工具。

更新

如果本隐私政策有任何更改，我们会在新版本发布时通知您。
